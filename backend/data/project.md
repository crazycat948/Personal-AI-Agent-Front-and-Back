# Texas Instruments AI-Based Unit Test Coverage Tool

Type: Capstone / Senior Design Project  
Collaboration: Texas Instruments

Description:

This project involves developing an AI-assisted tool that helps engineers analyze unit test coverage for C/C++ codebases.

Key Features:

- Parses C/C++ source files to extract function signatures, branches, and logical paths
- Detects potentially untested sections of code
- Uses OpenAI API to generate suggestions for additional unit test cases
- Automates part of the test coverage analysis workflow

Impact:

The tool aims to improve test coverage review efficiency and reduce manual effort during software testing.

Team Collaboration:

- Worked in a team of five students
- Weekly meetings with Texas Instruments engineering mentors
- Collaborative design, documentation, and iterative development process

Technologies Used:

- Python
- C/C++ parsing
- OpenAI API
- Git
- Linux development environment

# SkillVault — Full-Stack Skill Tracking Platform

Type: Personal Full-Stack Project

Description:

SkillVault is a platform designed to help users track personal skill development over time.

Users can record new skills, associate them with projects, and visualize long-term learning progress.

Key Features:

- Record and organize technical skills
- Link skills to projects
- Track progress and milestones
- Visualize long-term growth

Technical Architecture:

Backend:
- Java
- Spring Boot
- RESTful APIs

Database:
- MySQL
- relational schema design

Frontend:
- HTML
- CSS
- JavaScript

Goals of the Project:

This project was designed to demonstrate full-stack development ability, including backend architecture, database design, and front-end integration.
